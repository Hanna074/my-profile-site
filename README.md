# my-profile-site

Git 기능 실습을 수행하기 위한 자기소개 웹페이지 프로젝트입니다.
HTML, CSS, JavaScript를 활용하여 자기소개 페이지를 구성하고, 버튼 클릭 기능과 프로필 사진 표시 기능을 구현하였습니다.


## 구성
- index.html : 메인 웹페이지
- css/style.css : 웹페이지 스타일
- js/script.js : 버튼 클릭 기능
- images/profile.jpg : 프로필 사진 파일

## 주요 기능
- 자기소개 정보 표시
- 버튼 클릭 시 인사말 출력
- 프로필 사진 표시
